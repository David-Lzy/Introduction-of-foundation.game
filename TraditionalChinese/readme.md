# MIT2 / HuaXia 新人發育手冊（繁體）

> 學習路徑索引：[`INDEX.md`](./INDEX.md)

## README

在開始閱讀本手冊之前，請確認你已經滿足以下條件：

* 能源核心等級 ≥ 8
* 對遊戲的基本系統已有初步理解
* 有意願繼續長期發育，而不是短期體驗

本手冊並非「從零開始教學」。

如果你是第一次接觸《Foundation: Galactic Frontier》，或者仍處於探索基礎功能階段，建議先完成遊戲內的新手引導，並熟悉官方資料。

進入方式如下：

點擊遊戲左上角頭像
→ 選擇「銀河百科全書」
→ 閱讀相關基礎內容

銀河百科全書已經覆蓋：

* 戰力構成（旗艦 / 戰機 / 英雄 / 科技 / 英傑 / 建築）
* 基礎資源說明
* 戰鬥機制
* 建築與母艦系統
* 商會與星圖系統

本手冊的目標是：

幫助已經進入核心發育階段的玩家
避免資源浪費
理解活動節奏
建立正確的優先級思維

如果你還沒有達到能源核心 8 級以上，建議先專注完成基礎建設，再閱讀本內容。

遊戲是一個長期系統。
發育比衝動重要。
節奏比爆發重要。

歡迎來到真正的發育階段。
