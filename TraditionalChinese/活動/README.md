## 閱讀建議

- 想看商會 PVP 排班與搶點：先讀暗域戰場
- 想看商會護送與冷卻管理：先讀捍衛商路
- 想看士氣、預約與協作 PVE：先讀軍備競演
- 想看 VS 週六高強度作戰：先讀對決戰場

## 注意事項

- 活動規則可能隨版本與伺服器進度調整
- 涉及冷卻、報名、開放時間時，始終以遊戲內介面為準
- 商會類活動最怕的不是不會打，而是亂開、遲到和不統一
