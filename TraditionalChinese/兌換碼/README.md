# 兌換碼

本文檔提供《Foundation：銀河前線》的兌換碼信息。

## 使用說明

1. 點擊遊戲左上角頭像
2. 進入「設置」
3. 選擇「賬戶」標籤
4. 點擊「兌換禮包碼」
5. 輸入兌換碼並確認
