# 2026-05-14 官方 Windows 版上線與登入獎勵

## 當前確認的官方資訊

- 官方頁面已明確寫明 **Foundation：銀河前線** 已「Now officially live on Windows!」，並提供官網 Windows 安裝包下載入口。
- 官方頁面寫明：在 Windows 端登入可領取 **Permanent Portrait Frame**。
- 官方頁面還寫明：透過累計登入還能繼續領取更多獎勵，但頁面沒有公開逐檔獎勵明細。

## 對玩家的實際意義

- 想拿 **Permanent Portrait Frame** 的玩家，至少要完成一次 Windows 登入。
- 準備改用 PC 操作的玩家，現在已有官網直達的 Windows 安裝入口，不必依賴第三方模擬器頁面。
- 「累計登入更多獎勵」已確認存在，但因為官方頁面沒有放獎勵表和截止時間，實際檔位仍要以遊戲內活動頁為準。

## 建議動作

- 需要外觀獎勵的成員，盡快在 Windows 端完成首次登入。
- 想繼續拿累計獎勵的成員，後續連續登入前先查看遊戲內活動頁的獎勵檔位與結束時間。
- 下載入口優先使用官網 Windows 頁面，不要先參考第三方鏡像或模擬器導流頁。
