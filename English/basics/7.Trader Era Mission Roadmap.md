# Trader Era Mission Roadmap

The easiest mistake with this system is thinking it is just another reward board.

> **It is really a server progression roadmap.**

Treat Trader Era as a staged, server-wide mission track.
If the mission is cleared in time, you get the reward.
Even if it is not cleared, many systems still unlock when the phase ends.

---

## 1) What Trader Era actually is

### A server-wide phase mission system

Trader Era releases missions in phases.
Some focus on personal growth, some on guild occupation, and some on state-wide kills or Merit totals.

Core traits:

- **The missions are progressed by the whole server / state**
- Rewards are paid when the phase ends and the target is met
- **Many functions still unlock even if the target is missed**

So the practical reading is:

> This is not a side panel to ignore.
> It directly affects what your server can access next.

---

## 2) Where to check it

### Left-side entry in space view

The most common entry is on the **left side of the space screen**.
It sits with Counterattack Operation, and you can switch tabs to see the current Trader Era phase.

### Chronicle Gallery also works

If you prefer checking systems from the Mothership side, you can also open it through **Chronicle Gallery**.

Best habit:

> Check it whenever a new phase starts.

---

## 3) Why this matters more than the reward itself

### Early systems are unlocked on the phase line

The valuable part of the source page is not just the reward list.
It is the **unlock order**.

Examples from the page:

- **Mission 1** pushes toward **Starports** and **Tribute Ships**
- **Mission 3** pushes toward **Ruins Raiders**
- **Mission 4** unlocks the **guild ranking feature**
- **Mission 5 / 9 / 17** progressively unlock **Warp Relay**-related functionality
- **Mission 17** removes the **guild join restriction**
- **Mission 20** adds **end-point rewards** and advances higher-tier system content

Practical meaning:

- If a system feels unavailable, it may be because **your server phase has not reached it yet**
- Guild scheduling should follow the current phase instead of planning around content that is still locked

---

## 4) The mission flow worth remembering

You do not need to memorize all 22 missions.
You do need the broad structure.

### Early phase: build base growth and guild activity together

The early missions mainly push:

- Trader energy core levels
- Tribute Ship kills
- low-level planetary starport occupation

So early progression is not purely solo.

> **The system wants the whole state to move personal growth, guild activity, and PvE at the same time.**

### Mid phase: PvP and occupation pressure rises

The middle missions start asking for:

- higher-level planetary / stellar starport occupation
- state-wide Merit totals
- guild ranking thresholds

If you still play like a solo account here, you usually fall behind the server tempo.

### Late phase: higher-level systems and end-point rewards

Later missions move toward:

- **Lv.8 / Lv.9** content
- higher-level **Supreme Prayer Ship** access
- harder occupation targets
- extra **end-point rewards**

That makes Trader Era a good shorthand for your server's late-game roadmap.

---

## 5) The most useful mindset for ordinary players

### Do not stare only at the personal rewards

Most phases on the page repeat similar reward types, mainly:

- **Credits**
- **Far Voyage Memories**
- **Universal Speedups**

The more important question is:

> **What is this phase forcing the server to train right now?**

### Align your daily routine with the current phase

If the phase is pushing:

- Tribute Ships
- planetary starport occupation
- Trader-related growth

then your AP spending, guild coordination, and online window should lean that way.

That gives you two layers of value:

- you collect the phase reward
- your server reaches later systems faster

### Guild leadership should treat it like a scheduling signal

For leadership, Trader Era is not decorative.
It should directly shape:

- whether today's priority is occupation or Tribute Ship clearing
- which members need to be online for the phase target
- which events are worth organizing in the current server stage

## 6) Milestones worth remembering

### Mission 1

The page lists a requirement around raising a set number of Trader energy cores to **Lv.9**.
It also pushes out **Starports** and **Tribute Ships**.

That ties together:

- personal building growth
- guild external play
- active resource farming

very early.

### Mission 6

This phase asks the state to accumulate a set amount of **Merit** from Trader PvP.

Meaning:

> By this point, the system is clearly pushing the server into open conflict tempo.

### Mission 17

This removes the **guild join restriction**.

That matters for late joiners, guild transfers, recruitment, and restructuring.
If leadership plans roster moves, check whether the server has reached this point.

### Mission 20

This pushes higher-tier content and adds **end-point rewards**.

Usually that means:

- the server is already in a fuller late-game cycle
- execution quality matters more from here onward

---

## 7) Practical execution advice

### For ordinary players

1. Check the mission whenever the phase changes
2. Align your daily routine with the current target
3. If the phase is pushing occupation or Tribute Ships, try to do it with your guild

### For guild leadership

1. Post a short mission summary at the start of each phase
2. State clearly what should be pushed that day
3. Avoid event scheduling that is completely misaligned with the phase target

---

## Summary

Trader Era matters because it tells you:

1. **where your server currently is**
2. **what is unlocking next**
3. **what is most worth doing today**

The safest approach is not memorizing all 22 missions.
It is asking one question whenever the phase changes:

> **What is this phase forcing us to prioritize?**
