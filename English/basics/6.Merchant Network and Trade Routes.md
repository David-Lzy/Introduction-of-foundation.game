# Merchant Network and Trade Routes

These two systems are both steady daily income loops.
The difference is simple:

- **Merchant Network** spends **Action Points (AP)** on timed requests
- **Trade Routes** send fleets on long transports, with optional investment and hacking

---

## 1) Merchant Network: clear the timed requests first

### Unlock condition

- Clear **Counterattack Operation 5-5**
- Build the **Trader Network** facility

### Refresh schedule

The source page lists these refresh times in Japan time:

- **03:00**
- **09:00**
- **15:00**
- **21:00**

For actual play, use the in-game reset / red-dot timing as the final authority.

### Core routine

1. Check request color / quality first
2. Take the **highest-quality requests you can clear reliably**
3. After finishing, **tap again to claim**; completion is not fully automatic
4. After enough daily clears, remember to collect the **Home Port dividend**

### Common request types and AP cost

- **Planet landing / ground battle**: 5 AP
- **Supreme Vanguard**: 10 AP
- **Tribute Ship**: 30 AP
- **Supreme Prayer Ship**: 0 AP (co-op)
- **Commerce Guild support**: 5 AP
- **Exploration / chest**: 5 AP
- **Choice event**: 5 AP

### Practical advice

- Don’t leave **30 AP** Tribute Ship requests until you are nearly empty on AP
- Higher quality is better only if you can actually clear it
- When your **Commerce Guild** is active, do co-op dependent requests earlier so they don’t get stranded

---

## 2) Trade Routes: clear Priority Orders every day

### Unlock condition

- **Energy Core Lv.12**

### Two order types

- **Normal Orders**: unlimited per day, lower reward
- **Priority Orders**: limited per day, much better reward

### Priority Order count

This scales with **Trader Network** level:

- Lv.1: **1 per day**
- Lv.8: **2 per day**
- Lv.15: **3 per day**
- Lv.21: **4 per day**

### Core routine

1. Finish all **Priority Orders** first
2. Within one batch, pick the order with the best reward-to-time balance
3. Start long transports before sleep or long offline windows
4. If the fleet is suddenly needed elsewhere, **pause transport** and resume later

### Important details

- A fleet on trade duty is occupied and cannot do other commands at the same time
- Trade continues while you are offline
- Higher-quality order backgrounds usually mean more reward and longer duration

---

## 3) Investment and hacking: useful, but don’t click blindly

### Investment

- Target: a **Commerce Guild** member’s transport fleet
- Cost: **Galactic Coin**
- Return: when their transport finishes, you receive a **random share** of that shipment’s rewards

Recommended use:

- Prioritize active members who regularly run good orders
- If **Galactic Coin** is tight, secure your own key growth first

### Hacking

- Can only target **other players’ Priority Order transports**
- **Cannot** be used on the same **Commerce Guild**
- Success steals part of the transported cargo

Recommended use:

- Check your server / guild diplomacy rules first
- If you are going to hack, focus on high-value Priority Orders, not low-value scraps

---

## 4) A practical daily sequence

### Casual players

1. Log in and clear red-dot claims
2. Run one round of high-value Merchant Network requests
3. Start Priority Orders
4. Before logging off, check whether trade fleets need to be freed for combat

### Active players / officers

1. Track the four request refresh windows
2. Assign co-op tasks first
3. Then handle Priority Orders, investment, and any intentional hacking
4. Start the longest high-yield transport before offline time

---

## 5) Common ways players lose value

- Looking only at red dots and ignoring **quality / AP cost**
- Spending fleets elsewhere before finishing daily Priority Orders
- Forgetting that transport fleets are occupied when a rally suddenly starts
- Over-investing and slowing personal growth
- Forgetting to **manually claim** request rewards

---

## Summary

The core structure is:

- **Merchant Network** rewards timing discipline
- **Trade Routes** reward priority management and offline planning

Get the high-quality requests and daily Priority Orders stable first, then optimize investment, hacking, and finer routing.
