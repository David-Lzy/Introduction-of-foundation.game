# Core Resource Acquisition

Covers water, metal, food, and galaxy coins.

Understanding where resources come from is more important than focusing only on spending.

---

## 1) Merchant Network (base source)

Location: lower-left area, star-flag icon.

Recommendation:

> Complete as many tasks as possible.

For harder tasks (tribute ships, rallies), if current zone is too hard:

- Fly manually to lower-level sectors
- Kill lower-level targets there

Task checks completion, not target level.

---

## 2) Trade routes (depends on starport level)

Trade return scales with starport level.

Use galaxy map to check home port and station location.

General recommendation:

- Keep home port and station coordinated
- Move to higher-level free planets when possible (commonly Lv5 in current environment)

After migration, trade and merchant-network rewards improve.

---

## 3) Map-layer understanding

Normal UI is interaction layer.
Galaxy view is strategic layer.

Many targets/features are clearer in galaxy view.
Don’t rely only on auto-jump.

---

## 4) Info Data (easy to neglect, but valuable)

Info Data is not passive farm income.
It is closer to a utility currency earned by doing the right side content.

### Main source 1: Robot Explosion

If **Robot Explosion** is active, try to clear it at least once per run window.

Why it matters:

- First clear gives Info Data-type rewards
- Higher robot-kill counts in a run add extra reward tiers

This is a resource category you should not ignore completely.

### Main source 2: Space Encounter

`Space Encounter` is the more repeatable supplement.

Core flow:

1. Find an encounter request in space
2. Finish the dialogue / escort / combat objective
3. Unlock nearby chests and gathering points
4. Take Info Data from the unlocked loot

The important part is not only the request reward itself, but the **chests and higher-quality gathering points unlocked after completion**.

### Practical Space Encounter approach

- Clear encounters when you see them instead of tunnel-visioning only on mainline tasks
- If it turns into a space fight, protect the allied **Flagship** and remove the nearest enemies first
- Don’t camp one location forever; the same spot is not guaranteed to respawn the same way

It does not feel like a strict “limited daily attempts” system, but it is also not a guaranteed fixed-location loop.

### How to spend Info Data

Info Data is mainly spent in the **Info Store**.

Practical recommendation:

- Check discounted items at least once per week
- Buy for your real progression bottleneck first
- Don’t spend just because the currency is available

If you have a clear weekly shortage, discounted store slots are usually better than blindly converting into generic resources.

---

## Summary

Loop:
Merchant Network → Trade → Better starport level/location → Higher returns → Faster progression.

Common mistakes:

- Give up tasks when blocked
- Never relocate to better starports
- Ignore map-layer tools
- Ignore Robot Explosion / Space Encounter entirely and let Info Data income dry up
