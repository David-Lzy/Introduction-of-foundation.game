# Champion Weapon System

## System Overview
- **System Name**: Champion Weapon
- **Unlock Requirement**: Unlocks 90 days after server launch
- **Access Point**: Eternal Forge

## Features
- Equip Champions with exclusive weapons
- Forge and upgrade weapons via Eternal Forge
- Enhance Champion combat capabilities

## Unlock Timing
- New server players must wait 90 days from server launch
- Older server players can access immediately if server is 90+ days old
- Access Eternal Forge from the main interface

## Related Tasks
- New Champion Weapon point tasks added to Commerce Guild Duel event
- Complete tasks to obtain weapon enhancement materials
