# Champion Weapon System

## System Overview
- **System Name**: Champion Weapon
- **Unlock Requirement**: Unlocks on the Monday of Week 12 of the server, replacing the older "90 days after server launch" wording
- **Access Point**: Eternal Forge

## Features
- Equip Champions with exclusive weapons
- Forge and upgrade weapons via Eternal Forge
- Enhance Champion combat capabilities
- Each Champion can acquire and upgrade a signature armament, with weapon entries unlocked as the system develops

## Unlock Timing
- New servers should plan around **Week 12 Monday**, not a simple 90-day timer
- Older servers can access the feature if their server progress has already reached the unlock point
- Access Eternal Forge from the main interface

## Related Tasks and Scoring
- Commerce Guild Duel includes Champion Weapon-related point tasks
- Consuming **Weapon Prisms** grants Commerce Guild Duel points on Tuesdays and Fridays
- Points from **Weapon Energy Cores** were reduced in the official 1.1.21 version notes
- Matchmaking is optimized based on Champion Weapon unlock progress

## Officer Notes
- Before Tuesday / Friday Commerce Guild Duel scoring windows, confirm the live point values in-game
- If your guild is near Week 12, warn members not to over-plan around the old 90-day wording
- Treat the in-game event panel as final because live-service scoring can change by server progress
