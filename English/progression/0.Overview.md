# Progression System Overview

Progression is the long-term core of the game.

Resources determine speed.
Progression determines ceiling.

This chapter introduces main progression methods and topics covered in subsequent sections.

---

## I. Main Quest System

Main quests are:

> One-time resource and growth rewards.

Different from daily recurring tasks.

Completed quests don't refresh.

Better classified as "progression advancement" rather than daily resource farming.

---

### 1. Counterattack Operation

Location:

Top-left → Counterattack Operation

Features:

* Chapter-based progression
* Reputation level increases
* Unlocks additional bonuses

Affects:

* Supreme Prayer food bonuses
* Daily chest reward improvements

Not just story system,
but also long-term benefit enhancement system.

---

### 2. Activity System (Daily Progression)

Location:

Bottom-left → Activity

Complete daily tasks to earn activity points.

Reach different thresholds to claim rewards.

Progressive reward system.

---

### Activity Milestones (Paid Enhancement)

Activity system includes:

* Free rewards
* Additional paid reward track

When activated, each reward claim provides extra benefits.

Light spending enhancement option.

Covered separately in "Spending" chapter.

Not expanded here.

---

## II. Ship Progression System

Ship progression has two layers:

1. Flagship system
2. Building system

Different structures, both core growth components.

---

### 1. Flagship System

Location:

Bottom-right menu
Fourth icon from right (Flagship)

Can develop:

* Flagship level
* Component system

Components include:

* Radar
* Central Control Grid
* Railgun
* Composite Armor
* Thruster

Directly affects combat capability.

Detailed explanation in later sections.

---

### 2. Building System

Location:

Bottom-right → Foundation icon

Shows all buildings.

Buildings determine:

* Production capacity
* Feature unlocks
* Combat enhancements

Long-term investment progression content.

Priority planning crucial.

Detailed explanation later.

## Summary

Progression is not a single action.

It includes:

* Main quest advancement
* System unlocks
* Power development
* Long-term planning

Resources are tools.
Progression is direction.

Understanding structure prevents blind investment.
