# Redeem Codes

This document provides redeem code information for Foundation: Galactic Frontier.

## How to Redeem

1. Tap your profile icon (upper-left corner)
2. Go to Settings
3. Select Account tab
4. Tap Redeem Gift Code
5. Enter the code and confirm
