# Dominion Warzone Event

## Event Overview
- **Event Name**: Dominion Warzone
- **Original Test Phase**: Pioneer Test
- **Known Expansion**: expanded from Servers 1,001–1,010 to Servers 1,001–1,080 in version 1.1.11
- **Event Type**: Large-scale PvP competition

## Event Features
- Fight for glory and compete across eligible servers
- Large-scale PvP rules may vary by test / rollout stage
- Later version notes added preparation-stage tuning and a companion event

## Current Operational Notes
- Version 1.1.11 expanded Dominion Warzone eligibility to Servers 1001–1080
- Version 1.1.21 lowered daily point requirements during the Preparation Stage
- Version 1.1.21 added **Warzone Mobilization**, launching together with Dominion Warzone
- The event interface now has improved display for Garrison Stronghold levels

## Officer Checklist
- Confirm whether your server is inside the active rollout range before promising participation
- Before Preparation Stage begins, check the current daily point requirements in-game
- Watch for the Warzone Mobilization panel when Dominion Warzone opens
- Treat official version notes as directionally useful, but use the live in-game event panel as final
