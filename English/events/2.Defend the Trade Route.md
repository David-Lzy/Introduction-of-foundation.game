# Defend the Trade Route

> Audience: MIT2 / HuaXia officers and members  
> Role: manually started guild escort PvE event  
> Updated: 2026-05-02

## One-line summary

**Defend the Trade Route should not be opened casually. It is a guild escort event that depends on cooldown tracking, attendance timing, and a clear officer responsible for the full run.**

---

## 1. Event role

This event revolves around:

- officers starting the event
- escorting the guild trade ship
- clearing hostile NPC fleets along the route
- earning rewards based on progress and contribution

It is primarily a **guild PvE escort event**, not a player-versus-player activity.

---

## 2. Why cooldown management matters most

The current material suggests this is a **manual guild event**, not a fixed automatic schedule.
That means the most important management questions are:

- is the cooldown finished?
- are core players online?
- does it clash with VS, Shadowfront, or other major guild duties?
- who is responsible for calling targets and wrapping up?

### Minimum tracking fields

| Field | Why it matters |
|---|---|
| last start time | helps track cadence |
| end time | helps estimate next window |
| live in-game cooldown | the real authority |
| assigned coordinator | prevents accidental double-starts |

> The current screenshots do not show an exact cooldown duration, so do not hardcode a number into guild rules.

---

## 3. Main event features

### 3.1 Escort flow

Everyone fights around the trade ship's route. The real goal is to keep the route advancing, not to pad individual performance.

### 3.2 Contribution-based rewards

The event description implies stronger contributors receive better rewards. In practice that means:

- joining matters
- AFK hurts the whole run
- both high-power and lower-power members should contribute where they can

### 3.3 Settlement chests after the run

Public version notes mention **0-3 settlement chests** can appear after the event and may need to be collected manually near the ship's recent route.

MIT2 should therefore treat this as a required wrap-up step:

> **After the event, remind everyone to check for settlement chests and collect them manually.**

### 3.4 Flagship delegation and Lite Mode

If the live server version supports them, officers should also remind members about:

- Flagship delegation for players who cannot be online exactly on time
- Lite Mode for weaker devices

---

## 4. Expectations for members

### Everyone should

- watch the announcement instead of opening it at random
- prepare fleets 5-10 minutes early
- avoid sending main fleets away before the event
- repair serious damage before joining
- follow target calls when possible
- check for manual chest collection after the run

### High-power members should

- clear dangerous enemy fleets first
- help open the route ahead of the trade ship
- stabilize the pace for lower-power members
- help confirm leftover targets or chests at the end

### Lower-power members should

- stay near the main group
- clear safer targets or help finish weakened enemies
- use delegation/auto-participation if supported and requested

---

## 5. Recommended MIT2 operating flow

### Before opening

1. confirm cooldown
2. check for event conflicts
3. estimate attendance
4. assign one coordinator
5. send the notice
6. remind members to prepare fleets
7. remind delegation setup if the feature exists

### During the run

1. coordinator calls targets
2. high-power members clear high-threat enemies first
3. lower-power members follow the main direction
4. avoid mid-event arguments about distribution
5. suggest Lite Mode if lag becomes a problem

### After the run

1. remind members to check rewards
2. remind manual chest collection
3. log the event time and result
4. record the next available window
5. note whether the time slot worked well

---

## 6. Recommended internal rule

> Defend the Trade Route should only be opened by the guild leader or designated officers. Any officer who wants to start it should first confirm cooldown, attendance, and event conflicts.

Add one mandatory wrap-up line as well:

> After the event, the coordinator must remind members to check the nearby area for settlement chests and collect them manually.
