# 2026-05-14 官方 Windows 版上线与登录奖励

## 当前确认的官方信息

- 官方页面已明确写明 **Foundation：银河前线** 已“Now officially live on Windows!”，并提供官网 Windows 安装包下载入口。
- 官方页面写明：在 Windows 端登录可领取 **Permanent Portrait Frame**。
- 官方页面还写明：通过累计登录还能继续领取更多奖励，但页面没有公开逐档奖励明细。

## 对玩家的实际意义

- 想拿 **Permanent Portrait Frame** 的玩家，至少要完成一次 Windows 登录。
- 准备改用 PC 操作的玩家，现在已有官网直达的 Windows 安装入口，不必依赖第三方模拟器页面。
- “累计登录更多奖励” 已确认存在，但因为官方页面没放奖励表和截止时间，实际档位仍要以游戏内活动页为准。

## 建议动作

- 需要外观奖励的成员，尽快在 Windows 端完成首次登录。
- 想继续拿累计奖励的成员，后续连续登录前先查看游戏内活动页的奖励档位与结束时间。
- 下载入口优先使用官网 Windows 页面，不要先参考第三方镜像或模拟器导流页。
