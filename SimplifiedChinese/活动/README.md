## 阅读建议

- 想看商会 PVP 排班与抢点：先读暗域战场
- 想看商会护送与冷却管理：先读捍卫商路
- 想看士气、预约与协作 PVE：先读军备竞演
- 想看 VS 周六高强度作战：先读对决战场

## 注意事项

- 活动规则可能随版本与服务器进度调整
- 涉及冷却、报名、开放时间时，始终以游戏内界面为准
- 商会类活动最怕的不是不会打，而是乱开、迟到和不统一
