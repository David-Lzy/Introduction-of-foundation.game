# 兑换码

本文档提供《Foundation：银河前线》的兑换码信息。

## 使用说明

1. 点击游戏左上角头像
2. 进入「设置」
3. 选择「账户」标签
4. 点击「兑换礼包码」
5. 输入兑换码并确认
