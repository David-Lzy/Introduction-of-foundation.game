# MIT2 / HuaXia 新人发育手册

## README

在开始阅读本手册之前，请确认你已经满足以下条件：

* 能源核心等级 ≥ 8
* 对游戏的基本系统已有初步理解
* 有意愿继续长期发育，而不是短期体验

本手册并非“从零开始教学”。

如果你是第一次接触《Foundation: Galactic Frontier》，或者仍处于探索基础功能阶段，建议先完成游戏内的新手引导，并熟悉官方资料。

进入方式如下：

点击游戏左上角头像
→ 选择「银河百科全书」
→ 阅读相关基础内容

银河百科全书已经覆盖了：

* 战力构成（旗舰 / 战机 / 英雄 / 科技 / 英杰 / 建筑）
* 基础资源说明
* 战斗机制
* 建筑与母舰系统
* 商会与星图系统

本手册的目标是：

帮助已经进入核心发育阶段的玩家
避免资源浪费
理解活动节奏
建立正确的优先级思维

如果你还没有达到能源核心 8 级以上，建议先专注完成基础建设，再阅读本内容。

游戏是一个长期系统。
发育比冲动重要。
节奏比爆发重要。

欢迎来到真正的发育阶段。
